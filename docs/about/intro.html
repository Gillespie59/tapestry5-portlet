Tapestry5 Portlet Overview

This portlet bridge allows you to run Tapestry5 web apps in a portal environment without worrying about the underlying portlet API or portlet concepts.

This library is based on a google project started by Markus Feindler, Le Xuan Trun and Kristina B. Taylor. This was the first portlet bridge for Tapestry 5.1.0.5.

Afterwards, Felix Scheffer did an update of this lib in order to make the bridge usable with Tapestry 5.2.6. (see google project for more details)

After a rework phase, here is a new version of this bridge available for Tapestry 5.2.6, 5.3.x and 5.4.x (see github project for more details)
